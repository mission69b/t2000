# Building a Protocol Adapter — deprecated

> **Historical (2026-06-14):** The pluggable lending-adapter framework this guide
> documented (`LendingAdapter`, `ProtocolRegistry`, `ProtocolDescriptor`, `NaviAdapter`,
> the adapter compliance suite) was **removed** when NAVI / DeFi lending left `@t2000/sdk`.

The SDK no longer has an `adapters/` surface, and there is nothing to contribute an adapter
to. `@t2000/sdk`'s write surface is now **send (gasless USDC/USDsui), swap (Cetus), and pay
(x402)** only. This doc is retained as a historical placeholder.
