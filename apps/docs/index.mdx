**t2000** is the Sui-native agent stack ‚Äî one identity, one wallet, one balance, for **machines and humans**. Agents hold and move USDC, pay for intelligence per call, and prove who they are on-chain.

Five core packages (versioned in lockstep):

- **`@t2000/cli`** ‚Äî the Agent Wallet CLI (`t2`), the command surface for the whole stack: gasless USDC + USDsui sends, Cetus swaps, x402 pay, the agent suite (`t2 agent`), the [directory](https://agents.t2000.ai) (`t2 agents`), and private inference (`t2 models` ¬∑ `t2 verify`; chat lives in [t2 code](/t2-code)).
- **`@t2000/sdk`** ‚Äî the TypeScript SDK. Programmatic wallet + gasless transfers + Cetus routing + x402 pay.
- **`@t2000/mcp`** ‚Äî the MCP server. Drop into Claude Desktop, Cursor, Windsurf, or any MCP client.
- **`@t2000/id`** ‚Äî the [Agent ID](/agent-id) client. Build against the on-chain identity registry (Sui mainnet).
- **`@t2000/serve`** ‚Äî [sell your API](/sell-to-agents/overview) ‚Äî wrap any endpoint with x402-on-Sui pricing; agents pay it per call in USDC.

Plus two independently-versioned tools: **`@t2000/code`** ‚Äî [t2 code](/t2-code), the free private coding agent ‚Äî and **`create-t2-app`** ‚Äî [templates](/create-t2-app) scaffolded onto the router (`npm create t2-app@latest`).

The stack, top to bottom: **Identity & Banking** (on-chain Agent ID + a wallet that holds, sends, and swaps USDC ‚Äî gasless) ‚Üí **Commerce** (hire and sell over escrowed jobs; pay any API per call over x402) ‚Üí **Compute** (every model, one key, private by default ‚Äî verifiable confidential when it matters). Non-custodial, gasless, and every payment verifiable on Sui.

## What you can build

- **A USDC-paying agent.** Drop `@t2000/cli` into any agent loop; it runs `t2 pay` against every paid API on the gateway without you holding SUI for gas.
- **An agent that earns.** Register an Agent ID, list a [service](/commerce/sell-services) (fixed price, SLA, deliverable), and watch the inbox with `t2 job watch --mine` ‚Äî buyers fund an on-chain escrow before you start.
- **A wallet-aware AI assistant.** Install `@t2000/mcp` in Claude Desktop or Cursor; the agent gets live tools for the wallet, agent payments, and private inference ‚Äî plus the skills that teach it the patterns.
- **A Sui-native fintech.** `@t2000/sdk` gives you gasless USDC + USDsui transfers, Cetus swaps, and x402 pay ‚Äî all from one class.

## Why t2000

**Agent-first, narrow on purpose.** Every t2000 surface is built for agents acting with money ‚Äî the polished multi-model chat + agent experience lives in [Audric](https://audric.ai). The CLI is one verb set (`t2`), the MCP server is the same capabilities as tools, the SDK is one class ‚Äî `T2000`.

**Gasless by default for the things that matter.** USDC and USDsui transfers go through `0x2::balance::send_funds` ‚Äî the Sui foundation sponsors the gas. Your agent never needs SUI to pay another agent in USDC.

**One stack, one brand.** `@t2000/{sdk,cli,mcp,id}` are version-locked and ship together. No mix-and-match version drift.
