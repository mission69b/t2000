```bash curl
  -d '{

```

```typescript OpenAI SDK
import OpenAI from "openai";
const client = new OpenAI({ apiKey: process.env.T2000_API_KEY, baseURL: "https://api.t2000.ai/v1" });
const res = await client.chat.completions.create({
  model: "zai/glm-5.2",
  messages: [{ role: "user", content: "Hello" }],
});
```

**`POST /v1/chat/completions`** · Auth: bearer key (or keyless via [x402](/agent-payments)).

## Body

| Field | Type | Description |
|---|---|---|
| `model` | string | **Required.** Model id — see [`GET /v1/models`](/api-reference/models) (e.g. `zai/glm-5.2`, `phala/glm-5.2`). |
| `messages` | array | **Required.** OpenAI chat messages (`role` + `content`). |
| `stream` | boolean | Stream `chat.completion.chunk` SSE events, terminated by `data: [DONE]`. |
| `stream_options` | object | `{ "include_usage": true }` → a final usage chunk (carries `x_receipt_id` for confidential calls). |
| `max_tokens` | number | Max output tokens. |
| `temperature` | number | Sampling temperature (0–2). |

Tools/function-calling, vision, and the other OpenAI chat parameters pass through for models that support them.

## Response

The standard OpenAI `chat.completion` object (`choices[].message.content`, `usage`, …). Confidential (`phala/*`) responses also return an **`x-receipt-id`** header — fetch + verify it (see [Verify a response](/confidential-ai/verify)).

```json
{
  "id": "chatcmpl-…", "object": "chat.completion", "model": "zai/glm-5.2",
  "choices": [{ "index": 0, "message": { "role": "assistant", "content": "…" }, "finish_reason": "stop" }],
  "usage": { "prompt_tokens": 9, "completion_tokens": 12, "total_tokens": 21 }
}
```

Billed per token from your credit balance at the model's `GET /v1/models` rate. See [Pricing](/models#pricing).
