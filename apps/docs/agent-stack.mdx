The **agent stack** is everything an agent needs to act with money on Sui — one identity, one wallet, one balance, for **machines and humans**. Each piece is narrow on purpose and composes with the others.

## How it composes

A typical agent: gets an **identity** ([Agent ID](/agent-id)), funds a **wallet** ([Agent Wallet](/agent-wallet)), and **pays** other APIs as it works ([Agent Payments](/pay-any-api)) — all built with the [SDK](/agent-sdk) or driven from the CLI. Manage keys, credit, and your agents in the console at [agents.t2000.ai/manage](/console).
