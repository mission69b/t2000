t2000 gives an agent (or a person) one identity, one balance, and one key that spans private inference, payments, and on-chain commerce. Here's the five-minute path. (New here? Start with the [Introduction](/).)

## Prerequisites

- A Sui address — the wallet creates one for you.
- ~$1 of USDC to send + a little SUI (~0.05) for gas on swaps. USDC/USDsui sends and x402 API payments are **gasless**.
- Node.js 18+. An MCP-compatible client (Claude Desktop / Cursor / Windsurf / Cline / Continue / Codex) if you want the agent to call the wallet directly.

## 1. Install + create a wallet

```bash
npm install -g @t2000/cli
t2 init                              # create a wallet (local keypair, 0600 perms)
```

Or paste the **one-prompt setup** into any AI client (Cursor, Claude Code, ChatGPT) — it fetches the read-only setup skill (config-only; never moves funds) and wires everything:

```text
Run curl -sL https://t2000.ai/skills/t2000-setup, and use the returned setup instructions to set up my Agent Wallet.
```

`t2 init` registers your [Agent ID](/agent-id) too (gas permitting), so you're discoverable out of the box. See [Authentication](/authentication) for keys + funding.

## 2. Your first private inference call

<CodeGroup>

```bash CLI
t2 chat "Explain x402 in one sentence." --model zai/glm-5.2     # streams
t2 chat --model phala/glm-5.2 "Summarize this privately"        # 🔒 confidential (GPU-TEE)
```

```bash curl
curl https://api.t2000.ai/v1/chat/completions \
  -H "Authorization: Bearer $T2000_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{ "model": "zai/glm-5.2", "messages": [{ "role": "user", "content": "Hello" }] }'
```

```json MCP
// Any MCP client (Cursor, Claude Desktop) with T2000_API_KEY in the server env:
{ "name": "t2000_chat", "arguments": { "prompt": "Explain x402 in one sentence." } }
```

</CodeGroup>

Get a key (or fund a wallet) at [platform.t2000.ai](https://platform.t2000.ai) — see [Authentication](/authentication). Browse models with `t2 models` or [`GET /v1/models`](/models).

## 3. Your first agent payment

Pay any [x402](/agent-payments) service per call in USDC — no key, gasless, straight from your wallet:

```bash
t2 pay https://x402.t2000.ai/t2000/v1/chat/completions \
  --data '{"model":"zai/glm-5.2","messages":[{"role":"user","content":"hi"}]}' \
  --max-price 0.10
```

## Troubleshooting

<AccordionGroup>
  <Accordion title="t2: command not found (after a successful install)">
    The install wrote the binary, but npm's global `bin` directory isn't on your shell's `PATH`. Confirm:

    ```bash
    NPM_BIN="$(npm prefix -g)/bin"
    ls -l "$NPM_BIN"/t2 "$NPM_BIN"/t2000      # do the symlinks exist?
    "$NPM_BIN/t2" --version                    # runs via full path? (package is healthy)
    case ":$PATH:" in *":$NPM_BIN:"*) echo "✓ on PATH";; *) echo "✗ NOT on PATH → $NPM_BIN";; esac
    ```

    If the full-path command prints the version but it's NOT on PATH, add the dir to your shell profile:

    ```bash
    # zsh
    echo "export PATH=\"$(npm prefix -g)/bin:\$PATH\"" >> ~/.zshrc && source ~/.zshrc
    ```

    Then open a **fresh terminal** and re-run `t2 --version`. (`t2000` is a built-in alias for `t2`.)
  </Accordion>
  <Accordion title="EACCES / permission errors on install">
    Don't `sudo npm install` — point npm's global prefix at your home dir instead:

    ```bash
    mkdir -p ~/.npm-global
    npm config set prefix ~/.npm-global
    echo "export PATH=\"$HOME/.npm-global/bin:\$PATH\"" >> ~/.zshrc && source ~/.zshrc
    npm install -g @t2000/cli
    ```

    Or install Node via [nvm](https://github.com/nvm-sh/nvm) / Homebrew, which keep the global prefix user-writable.
  </Accordion>
</AccordionGroup>
