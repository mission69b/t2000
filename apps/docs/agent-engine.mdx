---
title: "Agent Engine (retired)"
description: "The @t2000/engine package has been retired. Audric v3 composes the AI SDK directly over @t2000/sdk."
---

The **Agent Engine** package (`@t2000/engine`) has been **retired**.

Audric v3 composes the [Vercel AI SDK](https://ai-sdk.dev) directly over [`@t2000/sdk`](/agent-sdk) — there is no separate engine layer anymore.

The already-published `@t2000/engine@4.x` remains on npm for the frozen legacy Audric app, but it is no longer developed and is not part of the current t2000 stack.

If you're building an agent, start with the [Agent SDK](/agent-sdk), [Agent Wallet](/agent-wallet), and [Agent Payments](/agent-payments).
