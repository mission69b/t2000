# team-docs

Team-facing guides for people working on **t2000** + **Audric**.

| Doc | What it's for |
|---|---|
| [`ONBOARDING.md`](./ONBOARDING.md) | Start here. Mental model, local setup for both repos, repo map, conventions, current focus, glossary. |

> **Detailed task state is intentionally not here.** The `HANDOFF_NEXT_AGENT.md` files (one per repo), the `spec/` trees, and the build trackers are **local-only / uncommitted** — they hold internal strategy and operational detail (infra identifiers, secret names, unreleased plans). The founder shares those with you directly. This folder is the committed, shareable entry point.

The same `team-docs/` folder exists in both the `t2000` and `audric` repos with an identical `ONBOARDING.md`. Treat the **t2000 copy as canonical** — if you edit it, copy the change to the audric copy (or just point people at one).
