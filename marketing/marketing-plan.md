## Product Description

**t2000 — The first bank for AI agents**

AI agents can browse the web, write code, and talk to APIs. But they can't manage money. t2000 changes that.

**What is it?**
A bank account that AI agents use to manage money. On Sui.

**What can it do?**

- **Save** — Earn yield automatically across NAVI and Suilend
- **Send** — Pay by name: `t2000 send $10 to alice`
- **Invest** — Buy crypto and gold with portfolio tracking and P&L
- **Strategies** — Predefined portfolios or build your own
- **Auto-Invest** — DCA on any schedule — daily, weekly, monthly
- **Earn yield on investments** — Lend invested assets while keeping price exposure
- **Investment rebalance** — Move earning positions to better-rate protocols automatically
- **Borrow** — Instant credit against savings
- **Rebalance** — Move savings to the best yield automatically
- **Exchange** — Swap any pair via Cetus aggregator
- **Pay** — Agent pays API paywalls. No key. No human.
- **MCP** — Works with Claude, Cursor, Copilot, Codex. 23 tools. Zero config.
- **Safeguards** — Limits, caps, kill switch. Protection before autonomy.
- **Contacts** — Named address book. Send by name, not hex.
- **Margin** — Leveraged trading with USDC collateral (coming soon)

**Five accounts:**
| Account | Purpose |
|---------|---------|
| Checking | Send, receive — liquid USDC |
| Savings | Earn yield on stablecoins |
| Credit | Borrow against savings |
| Exchange | Swap any supported pair |
