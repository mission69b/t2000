## Product Description

**t2000 — A bank account for AI agents**

AI agents can browse the web, write code, and talk to APIs. But they can't manage money. t2000 changes that.

**What can it do?**

- **Save** — Earn yield across NAVI and Suilend
- **Send** — Pay by name: `t2000 send $10 to alice`
- **Invest** — Buy crypto and gold with P&L tracking
- **Strategies** — Predefined portfolios or build your own
- **Auto-Invest** — DCA on any schedule
- **Borrow** — Instant credit against savings
- **Rebalance** — Move to the best yield automatically
- **Exchange** — Swap any pair via Cetus
- **Pay** — x402 micropayments, no API keys
- **MCP** — Works with Claude, Cursor, Windsurf. 23 tools. Zero config.
- **Safeguards** — Limits, caps, kill switch.
- **Contacts** — Send by name, not hex.

**Five accounts:**

| Account | Purpose |
|---------|---------|
| Checking | Send, receive — liquid USDC |
| Savings | Earn yield on stablecoins |
| Credit | Borrow against savings |
| Exchange | Swap any supported pair |
| Investment | Buy, earn, track P&L |

**Install:**
```
curl -fsSL https://t2000.ai/install.sh | bash
t2000 init
```

## MPP Launch Content

### Key visual — the 5-line code snippet

This is the hero asset. Use on landing page, Twitter, and README.

```typescript
import { sui } from '@t2000/mpp-sui/server';

const mppx = Mppx.create({
  methods: [sui({ currency: SUI_USDC, recipient: '0xYOUR_ADDRESS' })],
});

export const GET = mppx.charge({ amount: '0.01' })(
  () => Response.json({ data: 'paid content' })
);
```

**Why it works:** Developers see "I can accept Sui USDC on my API in 5 lines" — instant understanding, no ambiguity. Show the client side too for the full picture:

```typescript
import { sui } from '@t2000/mpp-sui/client';

const mppx = Mppx.create({
  methods: [sui({ client: suiClient, signer: keypair })],
});

const response = await mppx.fetch('https://api.example.com/resource');
// That's it. If the API returns 402, mppx pays automatically.
```
